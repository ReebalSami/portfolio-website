### 1. B2B-Sales-Lead-Pipeline (Datalogue, Jun.–Dez. 2025)
- **Kategorie**: KI, NLP
- **Tech**: Python, Streamlit, Agno, OpenAI, Gemini, Pydantic, SQLAlchemy, PostgreSQL (3NF), Vektorsuche, BeautifulSoup, Oxylabs, Docker
- **Kennzahl**: ~50 % Reduktion des manuellen Aufwands
- **GitHub**: https://github.com/ReebalSami/b2b-sales-lead-multi-agent-pipeline-showcase
- **Beschreibung**: End-to-End-Multi-Agenten-Pipeline zur Automatisierung der B2B-Sales-Recherche bei Datalogue. Vier spezialisierte LLM-Agenten — Company Targeting, Insights, Contact Discovery, Outreach Drafting — arbeiten über gemeinsamen State auf Basis eines PostgreSQL-3NF-Schemas und einer generischen News-Crawler-Pipeline zusammen. Human-in-the-Loop-Feedback verbessert die Vorhersagequalität kontinuierlich.

### 2. Urban Farming Plant Health (Deep Learning FH Wedel, SS 2025)
- **Kategorie**: KI, Data Science, Computer Vision
- **Tech**: PyTorch, timm, ViT, DINOv2, ConvNeXt, ResNet-50, EfficientNet, Grad-CAM, Attention Rollout, Streamlit, Python
- **Kennzahl**: 99,9 % / 99,7 % / 99,1 % pro Stufe
- **GitHub**: https://github.com/ReebalSami/urban-farming-plant-health-showcase
- **Beschreibung**: Hierarchische 3-Stufen-Pipeline (PlantID → Binäre Gesundheit → Krankheitsklassifizierung) auf 31.616 kuratierten Bildern aus mehreren öffentlichen Quellen. 3 Pflanzenarten, 16 Klassen. Vergleich von 8+ Architekturen über 3 Tiers. Wichtige Erkenntnis: die Trainingsdaten-Deploymentdomänen-Lücke (Modellverschlechterung auf Ganzpflanzenszenen gegenüber isolierten Blättern).

### 3. Insolvenzfrühwarnung (FH-Wedel-Seminar, WS 2024/25)
- **Kategorie**: Data Science
- **Tech**: Python, scikit-learn, XGBoost, Random Forest, Elastic Net, Makefile, Ökonometrie
- **Kennzahl**: AUC 0,78 – 0,86 über H1–H5-Horizonte
- **GitHub**: https://github.com/ReebalSami/seminar-bankruptcy-prediction
- **Beschreibung**: Multi-Horizont-Frühwarnsystem für Unternehmensinsolvenz. Horizontspezifische Modelle (H1–H5) auf dem Polish Companies Bankruptcy Dataset (43.004 Beobachtungen, 64 Finanzkennzahlen, 4,84 % positive Klasse).
