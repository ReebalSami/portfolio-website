### 2. Urban Farming Plant Health (FH Wedel Deep Learning, SS 2025)
- **Category**: AI, Data Science, Computer Vision
- **Tech**: PyTorch, timm, ViT, DINOv2, ConvNeXt, ResNet-50, EfficientNet, Grad-CAM, Attention Rollout, Streamlit, Python
- **Metric**: 99.9% / 99.7% / 99.1% per stage
- **GitHub**: https://github.com/ReebalSami/urban-farming-plant-health-showcase
- **Description**: Hierarchical 3-stage pipeline (PlantID → Binary Health → Disease Classification) across 31,616 curated images from multiple public sources. 3 plant species, 16 classes. Compared 8+ architectures across 3 tiers. Critical finding: the training-deployment domain gap (models degrading on whole-plant scenes vs. isolated leaves).
- **Highlights**: Hierarchical routing (+6pp over global classifier); data-quality pipeline removed ~30% irrelevant samples, perceptual hashing for dedup; 8+ architectures compared (ViT, DINOv2, ConvNeXt, ResNet, EfficientNet, MobileNet); both Grad-CAM and Attention Rollout explainability; domain-gap discovery shaped the future-work recommendations.
